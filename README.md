# CodeAlpha_Stock_Portfolio_Tracker# 📈 Stock Portfolio Tracker

## 📌 Project Overview

Stock Portfolio Tracker is a Python-based application developed as part of the CodeAlpha Python Programming Internship.

The program allows users to enter stock names and quantities, calculates the total investment value using predefined stock prices, and displays the portfolio's overall worth. The project demonstrates the use of dictionaries, user input, arithmetic operations, and file handling in Python.

---

## 🎯 Objective

To build a simple stock portfolio tracker that calculates the total investment value based on user-selected stocks and quantities.

---

## ✨ Features

✅ User inputs stock names and quantities

✅ Uses predefined stock prices

✅ Calculates total investment value

✅ Displays portfolio summary

✅ Supports multiple stock entries

✅ Optionally saves results to a text or CSV file

---
